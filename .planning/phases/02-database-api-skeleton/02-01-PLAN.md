# Phase 02 Plan 01: Database + Note CRUD API

## Objective

Add async MySQL (via asyncmy + SQLAlchemy 2), Alembic migrations, and a fully working
Note CRUD API to the FastAPI skeleton. Notes are not yet auth-gated (Phase 3). The
Docker Compose gains a mysql service; only the api port is exposed to the host.

## Context

- Phase 1 delivered: FastAPI skeleton, uv-managed deps, Dockerized, GET /health.
- Phase 2 success criteria (from ROADMAP.md): Note CRUD works in Docker via Swagger,
  pagination + sort + filter on GET /notes, Alembic runs the migration, tests pass.
- Stack decisions (CLAUDE.md / research): asyncmy (not aiomysql), SQLAlchemy 2.x
  DeclarativeBase, Alembic async env.py, python:3.12-slim, mysql:8.4.
